# Browser Automation Agent 

## Overview

This project is an **LLM-powered browser automation agent** that converts natural language instructions into executable browser actions.


## Tech Stack

- Python
- Playwright
- FastAPI
- Mercury LLM
- Requests

## Features

- Natural Language Browser Automation  
- Headless Browser Execution  
- Popup Handling  
- Retry Mechanism  
- FastAPI Integration  
- Screenshot Logging    

## Installation

pip install -r requirements.txt
playwright install
